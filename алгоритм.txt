Словесный алгоритм программы №1
Начало программы
Объявить три вещественные переменные a, b, c.
Ввести с клавиатуры значения a, b, c.
Проверить неравенство треугольника: если a + b ≤ c или a + c ≤ b или b + c ≤ a, вывести сообщение о несуществовании треугольника и завершить программу.
Присвоить логической переменной isRight значение false.
Проверить условие прямоугольности с допуском 1e-6: если модуль разности суммы квадратов двух сторон и квадрата третьей меньше 1e-6, присвоить isRight значение true.
Если isRight равно true, вывести сообщение, что треугольник является прямоугольным.
Если isRight равно false: а) Вычислить cosC = (a² + b² - c²) / (2·a·b). б) Вычислить angleC = arccos(cosC) · 180 / π. в) Вывести сообщение, что треугольник не является прямоугольным. г) Вывести значение angleC в градусах.
Завершить выполнение программы

Словесный алгоритм программы №2
Объявляется структура Stud, которая хранит: 
ФИО студента; 
группу; 
средний балл; 
указатель на следующий элемент списка. 
Пользователь вводит количество студентов n. 
Создаются два указателя: 
head — начало списка; 
tail — конец списка. 
В цикле от 1 до n: 
вводятся ФИО студента; 
вводится группа; 
вводится средний балл; 
создаётся новый элемент списка; 
данные записываются в новый элемент; 
если список пустой, новый элемент становится первым; 
иначе элемент добавляется в конец списка. 
После ввода всех студентов выполняется сортировка списка по среднему баллу: 
используются два вложенных цикла; 
сравниваются средние баллы студентов; 
если у первого студента балл больше, чем у второго, их данные меняются местами; 
в результате список сортируется по возрастанию среднего балла. 
Затем программа выводит отсортированный список студентов: 
номер студента; 
ФИО; 
группу; 
средний балл. 
После вывода памяти, выделенная под список, освобождается: 
поочерёдно удаляются все элементы списка. 
Программа завершает работу.
